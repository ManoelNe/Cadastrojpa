# Cadastrojpa
Projeto de cadastro usando Java, JPA/Hibernate e MySQL via XAMPP.
Cadastro JPA

Este é um projeto simples feito para estudar JPA e Hibernate.
Aqui eu aprendi a fazer um pequeno cadastro de pessoas usando Java e salvar os dados no banco de dados.

O que o projeto faz

Cadastra uma pessoa pedindo nome, email e idade pelo console.

Salva os dados no banco usando JPA.

Usa o XAMPP como servidor MySQL.

Tecnologias usadas

Java

JPA / Hibernate

MySQL (via XAMPP)

Maven

Objetivo

O objetivo deste projeto é praticar JPA, entender como funciona a configuração e aprender a conectar o Java com o banco de dados de forma mais simples do que usando Connection e ResultSet.
